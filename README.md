# Big Bark News & Media

The canine industry's dedicated news and media platform — built with Next.js 16, TypeScript, Tailwind CSS v4, Prisma 7 + SQLite, and NextAuth.

## Getting Started

```bash
npm install
npm run dev
```

Open [https://bbnm.ie](https://bbnm.ie) to view the live site, or [http://localhost:3000](http://localhost:3000) when running locally.

## Stack

- **Framework:** Next.js 16 (App Router)
- **Database:** SQLite via Prisma 7 + better-sqlite3
- **Auth:** NextAuth v4
- **Email:** Mailgun (EU region)
- **Styling:** Tailwind CSS v4
